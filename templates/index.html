About the Project

This project uses a Support Vector Machine trained on MRI images to distinguish between tumor and non-tumor scans. Deployed via Flask and accessible on all devices.

  • Python, Scikit-learn, Flask backend
  • HTML, CSS, JS frontend with image preview
  • Mobile-friendly upload from camera or gallery
  • Deployed on Render for live demo
